## Packages to install
### Neovim
Ubuntu
```
sudo apt install neovim
```
Arch
```
sudo pacman -S neovim
```
<hr>

### Zoxide
For Ubuntu, better to install via script
```
curl -sSfL https://raw.githubusercontent.com/ajeetdsouza/zoxide/main/install.sh | sh
```
Arch
```
sudo pacman -S zoxide
```
